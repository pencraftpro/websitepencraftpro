User Guide

Learn how to use PenCraft Pro with our detailed user guide.

Getting Started

This section will walk you through the basics of setting up PenCraft Pro for the first time.

  • How to install PenCraft Pro: Download and install PenCraft Pro. Follow the installation wizard for a seamless setup.
  • Creating your first note: Open the app, create a new note or new drawing by clicking the "Note" or "Drawing" button, and start typing or drawing right away.
  • Setting up cloud synchronization: You must log in to your PenCraft Pro account and enable cloud sync to securely store and access your notes across devices.

Note Creation and Organization

Learn how to create, edit, and organize your notes efficiently.

  • Using text formatting tools: Highlight text, add bullets, change font styles, and use bold/italic for better organization.
  • Adding images and sketches to your notes: Insert images or draw directly on your notes with the built-in stylus support.
  • Tagging and categorizing your notes: Use labels to group similar notes and easily find them in the search bar.

Drawing and Sketching

Discover the powerful drawing and sketching tools available in PenCraft Pro.

  • Choosing brush types and colors: Select from various brush types and colors to bring your sketches to life.
  • Using layers for advanced sketches: Add and manage layers to create complex drawings while keeping elements separate.
  • Exporting your sketches: Save your sketches as high-quality images or PDFs for sharing or printing.

Collaboration Features

Find out how to collaborate with your team using PenCraft Pro's real-time sharing and editing features.

  • Sharing notes with others: Share your notes via a secure link or invite collaborators directly from the app.
  • Real-time editing: Collaborate on notes with teammates in real-time and see changes as they happen.
  • Managing permissions for shared notes: Control who can view or edit shared notes by setting permissions.

Deleting Notes

Learn how to delete notes you no longer need while ensuring that important data isn't lost accidentally.

  • How to select notes for deletion: Select one or multiple notes from the list and click the "Delete" button to remove them.
  • Recovering accidentally deleted notes: Access the "Trash" folder to restore notes that were deleted by mistake.
  • Permanent deletion of archived notes: Permanently delete notes from the "Trash" folder to free up space.
